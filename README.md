# Native Logger

Very simple C# library to enable registering callbacks to native code to handle the log message directly. 
Used by other libraries to log in C# with optional forwarding to native. 

This library is used extensively in these following programs:
- **IMGBlibrary** - https://github.com/hectormiguel1/IMGBlibrary
- **WPDtool** - https://github.com/hectormiguel1/WPDtool/tree/native
